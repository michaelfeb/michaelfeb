## üìä GitHub Stats Widgets

### GitHub Readme Stats
```markdown
![GitHub Stats](https://github-readme-stats.vercel.app/api?username=michaelfeb&show_icons=true&theme=radical)
```

**Available Parameters:**
- `username` - Your GitHub username
- `show_icons=true` - Show icons
- `theme` - Theme name (see themes below)
- `hide=stars,commits,prs,issues,contribs` - Hide specific stats
- `count_private=true` - Include private contributions
- `include_all_commits=true` - Include all commits
- `line_height=25` - Adjust line height
- `title_color=fff` - Custom title color (hex)
- `text_color=fff` - Custom text color (hex)
- `icon_color=fff` - Custom icon color (hex)
- `bg_color=000` - Custom background (hex)
- `hide_border=true` - Hide border

## üåà Animated Elements

### Typing SVG
```markdown
[![Typing SVG](https://readme-typing-svg.herokuapp.com?font=Fira+Code&pause=1000&color=F75C7E&lines=First+line+of+text;Second+line+of+text)](https://git.io/typing-svg)
```

**Parameters:**
- `font` - Font name (URL encoded)
- `pause` - Pause duration in milliseconds
- `color` - Text color (hex without #)
- `lines` - Text lines separated by semicolons
- `center=true` - Center the text
- `vCenter=true` - Vertically center
- `width=435` - Width in pixels
- `height=50` - Height in pixels

### Header/Footer Wave

**Parameters:**
- `type` - Visual type
- `color` - Color or gradient
- `height` - Height in pixels
- `section` - header or footer
- `text` - Text to display
- `fontSize` - Font size

## üí° Pro Tips

1. **Keep it Fast**: Too many dynamic elements can slow loading
2. **Mobile Friendly**: Test how your profile looks on mobile
3. **Consistent Theme**: Stick to one color scheme/theme
4. **Update Regularly**: Keep your profile current with new projects
5. **Accessibility**: Ensure good contrast and alt text for images
6. **Legal**: Only use images and content you have rights to use
7. **Performance**: Some services may have rate limits

---

**Remember**: The best profile is one that reflects **you** - your skills, personality, and professional brand!
