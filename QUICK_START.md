## ‚ö° 5-Minute Setup

Follow these steps to personalize your profile quickly:

### Step 1: Edit README.md

Open `README.md` and replace these placeholders:

1. **Line 1**: Replace `[Your Name]` with your actual name
2. **Line 2**: Replace `[Your Role/Title]` and `[Your Location]`
3. **Lines 12-17**: Update the "About Me" section with your information
4. **Lines 24-29**: Update social media links (replace `yourusername`)

### Step 2: Choose Your Tech Stack

In the "Languages and Tools" section (starting at line 36):
- Keep the technologies you use
- Remove the ones you don't
- Add new ones from `BADGES_REFERENCE.md`

### Step 3: Update Featured Projects

At line 104, replace `project-name-1` and `project-name-2` with your actual repository names.

### Step 4: Commit and See Your Profile!

```bash
git add README.md
git commit -m "Personalize GitHub profile"
git push
```

Visit `https://github.com/michaelfeb` to see your new profile! üéâ

## üéØ Common Customizations

### Change Color Theme

Find any line with `theme=radical` and replace `radical` with:
- `dark`, `tokyonight`, `dracula`, `monokai`, `synthwave`
- Or any of the 50+ themes in `BADGES_REFERENCE.md`

Example:
```markdown
<!-- Before -->
![Stats](https://github-readme-stats.vercel.app/api?username=michaelfeb&theme=radical)

<!-- After -->
![Stats](https://github-readme-stats.vercel.app/api?username=michaelfeb&theme=tokyonight)
```

### Add a New Technology

Copy this template and replace the values:
```html
<a href="https://technology-website.com" target="_blank" rel="noreferrer">
  <img src="https://icon-url.svg" alt="tech-name" width="40" height="40"/>
</a>
```

Find icons at:
- [Devicon](https://devicons.github.io/devicon/)
- [Simple Icons](https://simpleicons.org/)

### Remove a Section

Don't want a particular section? Just delete everything between the `---` markers for that section!

## üÜò Need Help?

### Profile Not Updating?
- Clear your browser cache
- Wait a few minutes for GitHub to refresh
- Check that you're editing the correct file

### Stats Not Showing?
- Ensure your repositories are public
- Wait a few minutes for the API to update
- Check your username is correct

### Images Not Loading?
- Verify the image URLs are correct
- Check if the external service is operational
